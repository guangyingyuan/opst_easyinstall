# OpenStack Install Script

## Environment
 * OS: Ubuntu 15.10 Server (amd64) minimal
 * OpenStack: Liberty
 * All-in-one

## These scripts can install...
 * Keystone
 * Glance
 * Nova
 * Neutron
 * Cinder
 * Ironic

## How to Use

### Install Keystone
 1. `chmod +x install-keystone.sh`
 2. `sudo ./install-keystone.sh` (Need root priviledge)
